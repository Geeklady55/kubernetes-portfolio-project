Kubernetes Portfolio Project

Author: Colleen Cummings

This project demonstrates a local Kubernetes deployment using Docker, kind, kubectl, health checks, and troubleshooting workflows.

What this project shows

  • Containerized web app deployment
  • Kubernetes Deployment and Service
  • Observability concepts: logs, metrics, events
  • Health checks and scaling concepts

Useful commands

  • kubectl get pods
  • kubectl get svc
  • kubectl logs <pod>
  • kubectl describe pod <pod>
  • kubectl top pods
